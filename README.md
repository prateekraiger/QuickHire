## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+
- pnpm (recommended) or npm

### Installation

Install the dependencies:

```bash
pnpm install
# or
npm install
```

### Development

Start the development server with HMR:

```bash
pnpm dev
# or

```

Your application will be available at `http://localhost:5173`.
